Node Modules

Node modules are packages we use as developers to access a large variety of functionality. It can be from low-level working with files to a higher level library for building web application, databases, or user authentication. It is an easy and reliable way to add functionality to your application. Some of the node modules we have you used in class are:

Nodemon

This is a module we use to automatically restart our application when there are changes detected in the source code. It helps developers speed up the process by automatically restarting the server when you make changes. It is useful tool in the development stages of your application. Nodemon monitors the file system for any changes in the code and you can define it by writing this in a nodemon.json file: 

                
                    {
                        "ext": "js, json, html"
                    }
