Welcome to my Node.js Mandatory I

Node.js is a JavaScript runtime enviroment that lets you execute JavaScript code outside of a web browser. It is used by developers to build fast and high performance applications on the server side. Node.js has a large variety of modules and libraries for developers to use that makes it easy to add functionality to your application.
