Vercel

Here I will show you how you can host your website on a website like Vercel. It is super easy and shouldn't take long to setup:

First you will configure a vercel.json file in your project, so vercel knows how to run and route your application:

                
                    {
                        "version": 2,
                        "builds": [
                          {
                            "src": "app.js",
                            "use": "@now/node"
                          }
                        ],
                        "routes": [
                          {
                            
                            "src": "/(.*)",
                            "dest": "app.js"
                          }
                        ]
                      }

(Make sure your github is up to date with your project)

Then you can navigate to their website wwww.vercel.com and then you login in the right corner

I would prefer logging in with github as I use my github to pull the code I want to host.

When you are logged in you start by clicking on “Create new project” below in the center

Then you choose your git repository, you want to work from

Then you can configure your project. Name it what you want and you can also specify if you have worked with a certain framework.

What we will configure is what root, the project is in

When your project is fully configured, you can press “Deploy”. It will then deploy your website

Then you can go visit your page
